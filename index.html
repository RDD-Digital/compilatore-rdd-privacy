Compilatore RDD (Invitalia) — Estensione Chrome

Ultimo aggiornamento: 8 maggio 2026 · Versione 1.0

Questa informativa descrive come l'estensione Chrome Compilatore RDD (Invitalia) (di seguito, "l'estensione") tratta i dati dell'utente. Il rispetto della privacy è un valore fondamentale: l'estensione è progettata per minimizzare la raccolta di dati e per non trasferire informazioni a server gestiti dal titolare se non strettamente necessario al funzionamento.

1. Titolare del trattamento

Il titolare del trattamento dei dati è RDD Italia (di seguito "RDD" o "noi").

  • Sito web: https://www.rdditalia.com
  • Per esercitare i diritti previsti dal GDPR, vedi la sezione "Diritti dell'utente".

2. Quali dati l'estensione tratta

L'estensione tratta esclusivamente i seguenti dati, tutti forniti volontariamente dall'utente:

  • Chiave API OpenAI: una credenziale personale fornita dall'utente nelle impostazioni dell'estensione, necessaria per autenticare le chiamate ai server di OpenAI.
  • Documenti aziendali (PDF, DOCX) caricati dall'utente per estrarne il testo da utilizzare nella compilazione dei form. Possono contenere dati identificativi dell'azienda e di persone fisiche (nome, indirizzo, email, codice fiscale, partita IVA, ecc.).
  • Schema dei form Invitalia: nome, etichetta, tipo e opzioni dei campi presenti nella pagina del bando aperta dall'utente. Nessun valore già inserito nei campi viene letto.
  • Preferenze utente: modello AI selezionato, tema dell'interfaccia.

3. Dati che l'estensione NON tratta

  • Cookie del browser, sessioni, password salvate o credenziali SPID/CIE/CNS dell'utente.
  • Storage o dati di siti diversi da *.invitalia.it.
  • Cronologia di navigazione, posizione geografica, attività di mouse/tastiera al di fuori del widget.
  • Identificatori univoci di dispositivo, fingerprinting, analytics di utilizzo.

4. Dove vengono memorizzati i dati

I dati personali e i documenti restano esclusivamente nel browser dell'utente, salvati tramite l'API chrome.storage:

  • chrome.storage.sync: chiave API OpenAI, modello AI, preferenze. Se l'utente ha attiva la sincronizzazione di Chrome, questi dati possono essere replicati sui dispositivi sincronizzati dello stesso utente.
  • chrome.storage.local: documenti caricati (fino a 10 MB), cache temporanea. Non lasciano mai il dispositivo.

Nessun dato viene inviato a server di RDD Italia. RDD non ospita backend, non riceve copie dei documenti né delle chiavi API, non registra log di utilizzo.

5. Trasferimenti a terzi: OpenAI

Quando l'utente preme il bottone Compila, l'estensione invia i seguenti dati ai server di OpenAI, L.L.C. (USA), all'endpoint https://api.openai.com/v1/chat/completions:

  • Lo schema dei campi del form della pagina aperta;
  • Il testo estratto dai documenti che l'utente ha selezionato per la compilazione corrente;
  • Una richiesta di tipo "completion" autenticata con la chiave API dell'utente.

OpenAI è il fornitore del servizio di intelligenza artificiale. RDD non ha alcun ruolo di intermediazione in questo trasferimento: l'utente, configurando la propria chiave API, instaura un rapporto diretto con OpenAI. L'estensione si limita ad agire come client locale che facilita la chiamata.

Privacy policy di OpenAI: https://openai.com/policies/privacy-policy

Per le modalità di trattamento dei dati da parte di OpenAI (incluse le impostazioni "data retention" e "training opt-out") fare riferimento al pannello di controllo del proprio account OpenAI su platform.openai.com.

6. Finalità del trattamento

I dati sopra descritti sono trattati esclusivamente per le seguenti finalità:

  • Identificare i campi del form del bando Invitalia aperto dall'utente.
  • Generare suggerimenti di compilazione tramite OpenAI a partire dai documenti dell'utente.
  • Inserire i suggerimenti nei rispettivi campi del form.
  • Conservare le credenziali e i documenti tra una sessione e l'altra per evitare ricaricamenti ripetuti.

I dati non vengono utilizzati per profilazione, marketing, analisi commerciali, valutazione dell'affidabilità creditizia o altre finalità non correlate al funzionamento dell'estensione.

7. Base giuridica (GDPR)

Il trattamento si basa sul consenso esplicito dell'utente, manifestato dall'installazione dell'estensione, dalla configurazione della chiave API e dal caricamento volontario dei documenti. L'utente può revocare il consenso in qualsiasi momento disinstallando l'estensione e/o cancellando i propri dati come descritto al punto 9.

8. Conservazione dei dati

I dati restano nel browser dell'utente fino a quando l'utente stesso non li rimuove (eliminando i documenti dall'interfaccia del widget, revocando la chiave API o disinstallando l'estensione). RDD non conserva alcun dato.

I dati inviati a OpenAI sono soggetti alle politiche di conservazione di OpenAI.

9. Diritti dell'utente (GDPR Art. 15-22)

L'utente residente nell'Unione Europea può esercitare in qualsiasi momento i diritti previsti dal Regolamento UE 2016/679 (GDPR):

  • Accesso: tutti i dati trattati dall'estensione sono visibili nel widget e nelle impostazioni.
  • Rettifica: l'utente può modificare la chiave API, i documenti e le preferenze in qualunque momento.
  • Cancellazione: l'utente può eliminare singolarmente i documenti, sostituire la chiave API o disinstallare l'estensione (tutti i dati vengono cancellati). Per revocare anche la chiave API lato OpenAI vedi platform.openai.com/api-keys.
  • Portabilità: i documenti caricati restano nei file originali sul dispositivo dell'utente.
  • Reclamo all'autorità di controllo: l'utente può presentare reclamo al Garante per la protezione dei dati personali (garanteprivacy.it).

10. Sicurezza

L'estensione adotta le seguenti misure di sicurezza:

  • Storage locale tramite API native Chrome (chrome.storage), protetto dal sandbox del browser.
  • Comunicazione esclusivamente via HTTPS (TLS 1.2+) con i server di OpenAI.
  • Nessun server intermedio gestito da RDD: non esistono "punti di intercettazione" tra il browser dell'utente e OpenAI.
  • Codice dell'estensione disponibile su richiesta per audit di sicurezza.

11. Minori

L'estensione è destinata a un utilizzo professionale (compilazione di domande di finanziamento) e non è rivolta a minori di 16 anni. Non raccogliamo intenzionalmente dati di minori.

12. Modifiche a questa informativa

Eventuali modifiche a questa informativa verranno pubblicate sulla stessa pagina con aggiornamento della data in cima. In caso di modifiche sostanziali, gli utenti verranno informati al primo avvio successivo dell'estensione.

13. Contatti

Per qualsiasi richiesta relativa a questa informativa o ai propri dati, è possibile contattare RDD Italia tramite il sito ufficiale: https://www.rdditalia.com
